準備¶

Pythonを用いてWebスクレイピングするために必要な準備を実施します。

ターミナルの用意¶

ターミナルとは、コマンドを用いてMacの操作や設定を行うツールです。
Windowsでは、コマンドプロンプト、Windows Power Shelと同様と考えて大丈夫です。
Macでプログラミングを行う際は使用する必要があります。

ターミナルは、

「アプリケーションフォルダー」 → 「ユーティリティフォルダ」 → 「ターミナル.app」

にあります。

今回使用するコマンドは、

  • 使用コマンド
    • cd
    • pwd
    • mkdir
    • ls
    • touch

にまとめてあります。

これから今回の作業で必要な準備を実施していきます。 まずは、デスクトップに移動しましょう。

$ cd ~/Desktop/

移動したら、移動できているかどうかを確認します。

$ pwd
/Users/taku/Desktop

例のように/Users/{自分のユーザ名}/Desktopとなっていれば大丈夫です。 デスクトップに移動できたら今回使用するファイルを格納するディレクトリを作成します。

$ mkdir web

作成したら、作成できているかどうかを確認します。

$ ls
web

webというディレクトリが作成されていれば大丈夫です。 最後に作成したディレクトリに移動した後、移動できているかを確認します。

$ cd web
$ pwd
/Users/taku/Desktop/web

/Users/{自分のユーザ名}/Desktop/webになっていれば事前準備完了です。

Python3のインストール¶

python3.8.1のインストール package を用意したので今回はこれを用いてpython3をインストールしてください。

ダウンロードしたpkgファイルを実行し、Python3.8.1をインストールします。
基本的にはそのまま「続ける」ボタンの連打で大丈夫です。
